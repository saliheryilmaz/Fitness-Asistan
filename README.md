# 🏋️ Fitness Asistan

> **Yapay zeka destekli kişisel fitness ve beslenme takip uygulaması.**  
> Django REST API + Groq LLM entegrasyonu ile geliştirilmiş full-stack web projesi.

---

## 📌 Proje Hakkında

**Fitness Asistan**, kullanıcıların günlük kalori alımını, beslenme düzenini ve fitness ilerlemelerini takip edebildiği; aynı zamanda yapay zeka ile beslenme ve egzersiz önerileri alabildiği full-stack bir web uygulamasıdır.

Proje, gerçek dünya kullanım senaryoları gözetilerek tasarlanmıştır: kullanıcı kimlik doğrulaması, dinamik veri görselleştirme ve LLM tabanlı AI asistan entegrasyonu içermektedir.

---

## 🚀 Özellikler

### 🤖 Yapay Zeka Asistan
- **Groq API** entegrasyonu ile güçlendirilmiş gerçek zamanlı AI sohbet arayüzü
- Kullanıcının profili (yaş, kilo, hedef) bağlamında kişiselleştirilmiş beslenme ve egzersiz tavsiyeleri
- Doğal dil ile soru-cevap desteği

### 🥗 Kalori & Beslenme Takibi
- Günlük öğün kaydı (kahvaltı, öğle, akşam, ara öğün)
- Makro besin hesaplama: protein, karbonhidrat, yağ
- Hedef kalori belirleme ve günlük kota takibi

### 📈 İlerleme Takibi & İstatistikler
- Kullanıcıya özel ilerleme grafikleri (kilo, kalori, aktivite)
- Haftalık ve aylık performans özetleri
- Geçmiş veri analizi ile trend görselleştirme

### 🔐 Kullanıcı Kaydı & Kimlik Doğrulama
- Django'nun built-in authentication sistemi ile güvenli kayıt/giriş
- Profil yönetimi: yaş, boy, kilo, hedef belirleme
- Oturum bazlı kullanıcı verisi izolasyonu

### 🔌 REST API
- Django REST Framework ile tasarlanmış RESTful endpoint'ler
- JSON tabanlı veri alışverişi
- Frontend ile asenkron iletişim (Fetch API / AJAX)
